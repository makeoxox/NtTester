    函数
    • 函数名：Nt_Import(js)
    • 解释：导入外部脚本函数
    • 参数：外部脚本路径(String)
    • 返回值：无
    • 示例：Nt_Import("Project-Test/js/area.js")
  • 对象
    • 对象名：Nt_Asyn
    • 解释：进行异步处理的对象
    • 包含属性：无
    • 包含函数：
      • 函数名：run(func)
      • 解释：执行异步处理
      • 参数：异步回调函数(Funciton)
      • 返回值：无
      • 示例：Nt_Asyn.run(function(){ /* do someting */ })
    • 对象名：Nt_Ptr
    • 解释：输出到控制台的打印对象
    • 包含属性：无
    • 包含函数：
      • 函数名：println(content)
      • 解释：打印并换行
      • 参数：打印的内容(String)
      • 返回值：无
      • 示例：Nt_Ptr.println(".....")
      • 函数名：print(content)
      • 解释：打印不换行
      • 参数：打印的内容(String)
      • 返回值：无
      • 示例：Nt_Ptr.print(".....")
    • 对象名：Nt_Net
    • 解释：进行网络通讯的对象
    • 包含属性：
      • 属性名：tcp
      • 解释：tcp通讯对象
      • 包含函数：
        • 函数名：send(content,ip,port)
        • 解释：发送消息并等待返回
        • 参数：发送的消息内容(String)，ip地址(String)，端口号 (String)
        • 返回值：返回的消息(String)
        • 示例：var recv = Nt_Net.tcp.send("...","127.0.0.1","1551")
        • 函数名：sendNoRecv(content,ip,port)
        • 解释：发送消息不等待返回
        • 参数：发送的消息内容(String)，ip地址(String)，端口号 (String)
        • 返回值：无
        • 示例：var recv = Nt_Net.tcp.sendNoRecv("...","127.0.0.1","1551")
      • 属性名：http
      • 解释：http通讯对象
      • 包含函数：
        • 函数名：post(url,msg,code,contentType)
        • 解释：以post发送消息并等待返回
        • 参数：url（String）,发送的消息(String)，编码方式[String],内容类型[String]
        • 返回值：返回的消息(String)
        • 示例：var recv = Nt_Net.http.post("http://xxx.com/file","getfile","utf-8","text/plain")
        • 函数名：get(url,code)
        • 解释：以get发送消息并等待返回
        • 参数：url（String），编码方式[String]
        • 返回值：返回的消息(String)
        • 示例：var recv = Nt_Net.tcp.sendNoRecv("http://xxx.com/file","utf-8")
    • 包含函数：无
    • 对象名：Nt_Db
    • 解释：连接数据库对象
    • 包含属性：无
    • 包含函数：
      • 函数名：connect(url,driver,username,password)
      • 解释：连接到目标数据库
      • 参数：数据库URL(String),数据库java驱动名(String),数据库用户名(String),数据库密码（String）
      • 返回值：无
      • 示例：Nt_Db.connect("jdbc:sqlite://D:/NtData.db","org.sqlite.JDBC","kege","123456")
      • 函数名：query(sql,success)
      • 解释：查询数据库表
      • 参数：SQL查询语句(String),回调函数（Function）
      • 返回值：查询回的数据（Array）
      • 示例：var array = Nt_Db.query("select * from wsunit",function(array){ /* do something */ })
      • 函数名：execute(sql)
      • 解释：增删改数据库表
      • 参数：SQL变更语句(String)
      • 返回值：无
      • 示例：var array = Nt_Db.execute("insert into wsunit (unitid,name) values ('100001','ATMNumber1')")
    • 对象名：Nt_Control
    • 解释：数据显示控件对象
    • 包含属性：无
    • 包含函数：
      • 函数名：Table(data,title)
      • 解释：表格显示数组数据
      • 参数：数据数组（Array）,标题[String]
      • 返回值：无
      • 示例：Nt_Control.Table([{name:'kege',gender:'1'},{name:'wuyihong',gender:'2'},"员工表"])
      • 函数名：Graph(base64img,title)
      • 解释：显示Base64编码图片
      • 参数：Base64编码图片(String),标题[String]
      • 返回值：无
      • 示例：Nt_Control.Graph("base64 code...","身份证")
      • 函数名：Browser(content,contentType,title)
      • 解释：内置浏览器
      • 参数：显示内容可以是url或html代码(String)，指定内容类型url或html(String),标题[String]
      • 返回值：无
      • 示例：Nt_Control.Browser("http://www.baidu.com","url","浏览器")
    • 对象名：Nt_Json
    • 解释：Json处理对象
    • 包含属性：无
    • 包含函数：
      • 函数名：parse(jsonStr)
      • 解释：将json字符串转成Object或数组
      • 参数：json字符串(String)
      • 返回值:json对象(Object/Array)
      • 示例：var obj = Nt_Json.parse("{'name':'kege'}")
      • 函数名：stringify(json)
      • 解释：将json对象或数组转成json字符串
      • 参数：json对象或数组(Object/Array)
      • 返回值：json字符串（String）
      • 示例：var jsonStr = Nt_Json.stringify([{name:'kege'},{name:'wuyihong'}])
      • 函数名：json2xml(jsonStr,root)
      • 解释：将json字符串转为Xml字符串
      • 参数：json字符串（String）,指定一个根节点root(String)
      • 返回值：xml字符串（String）
      • 示例：var xmlStr = Nt_Json.json2xml("{'name':'kege'}","name")
      • 函数名：xml2json(xmlStr)
      • 解释：将Xml字符串转为json字符串
      • 参数：xml字符串（String）
      • 返回值：json字符串（String）
      • 示例：var jsonStr = Nt_Json.xml2json("<root><name>kege</name></root>")
  • 类
    • 类名：Nt_File
    • 解释：文件操作类
    • 构造函数：
      • 函数名：Nt_File(path)
      • 解释：使用文件路径构造文件对象
      • 参数：文件路径（String）
      • 返回值：文件对象（Nt_File）
      • 示例：var file = new Nt_File("D:/Ntlog/trace/xxx.log")
    • 包含函数：
      • 函数名：createNewFile()
      • 解释：在硬盘上创建该空文件
      • 参数：无
      • 返回值：无
      • 示例：file.createNewFile()
      • 函数名：delete()
      • 解释：在硬盘上删除该文件
      • 参数：无
      • 返回值：无
      • 示例：file.delete()
      • 函数名：exists()
      • 解释：判断硬盘是否存在该文件
      • 参数：无
      • 返回值：判断值（Boolean）
      • 示例：var flag = file.exists()
      • 函数名：getParent()
      • 解释：获得上层路径
      • 参数：无
      • 返回值：上层路径（String）
      • 示例：var path = file.getParent()
      • 函数名：isFile()
      • 解释：判断该文件是否是文件（不是文件夹）
      • 参数：无
      • 返回值：判断值（Boolean）
      • 示例：file.isFile()
      • 函数名：isDirectory()
      • 解释：判断该文件是否是文件夹（不是文件）
      • 参数：无
      • 返回值：判断值（Boolean）
      • 示例：file.isDirectory()
      • 函数名：listFiles()
      • 解释：返回该文件对象子文件数组
      • 参数：无
      • 返回值：文件数组（Array[Nt_File]）
      • 示例：var array = file.listFiles() ; array[0].exists()
      • 函数名：listFileNames()
      • 解释：返回该文件对象子文件名数组
      • 参数：无
      • 返回值：文件名数组（Array[String]）
      • 示例：var array = file.listFileNames() ; NtPtr.println(array[0])
      • 函数名：read(decode)
      • 解释：按编码读取文件内容
      • 参数：编码[String]
      • 返回值：文件内容（String）
      • 示例：var content = file.read("utf-8") ;
      • 函数名：write(content,encode,overwrite)
      • 解释：按编码写入文件内容
      • 参数：内容（String）,编码[String],是否覆盖[Boolean]
      • 返回值：无
      • 示例：file.write("hello","utf-8",true)
