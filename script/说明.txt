#所有脚本请使用GBK编码。

1.h.js中为导入库与自定义函数，只有在此脚本添加的库，才能同时在after.js和before.js中同时生效。

	第三方库请放在lib目录下，使用importPackge("包名")导入;
	
2.after.js为接受返回后处理数据脚本，其中入口函数名afterReceive。 
	示例：
		function afterReceive(data){
		
				（此处写自己的脚本）
		
			return data;
		}
		data :
			  描述 ：  收到的返回数据。
			   数据类型  ：String   
			  函数       :    字符串函数
		
3.before.js为发送数据前处理数据脚本，其中入口函数beforeSend。
	示例：
		function beforeSend(data){
		
				（此处写自己的脚本）
		
			return data;
		}
		data :
			  描述 ：  发送的数据。
			   数据类型  ：String   
			  函数       :    字符串函数
			  
4.调试脚本名任意，入口函数main，并且带一个入口参数。
	示例：
		function main(arg){
				（此处写自己的脚本）

			return null;

		}
